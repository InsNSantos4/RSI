Introdução à Programação

Objetivos
Desenvolver algoritmos para a solução de problemas específicos
Implementar programas e scripts utilizando uma linguagem de programação
Fazer depuração de programas informáticos

Resultados de Aprendizagem
No final da lecionação o estudante deve ser capaz de:

Desenvolver algoritmos para a solução de problemas específicos
Implementar programas e scripts utilizando uma linguagem de programação
Fazer depuração de programas informáticos
